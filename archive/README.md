# Archive

This directory keeps code and artifacts that are useful for provenance or future ablation, but are not part of the current main CompGen-GRPO training path.

- `legacy_scripts/`: older launch/debug scripts.
- `legacy_rewards/`: original GDino, GIT, and LLaVA ORM reward implementations.
- `legacy_dependencies/`: dependencies required only by legacy rewards, such as LLaVA-NeXT.
- `run_artifacts/`: local training outputs, logs, and TensorBoard events.
- `third_party_extras/`: unused third-party demo files, notebooks, Dockerfiles, and visual assets.

The active training path remains under `src/t2i-r1/src/`.
