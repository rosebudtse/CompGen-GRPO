# CompGen-GRPO 训练环境 requirements
#
# 适用环境（worker 节点）：
#   - Python 3.11
#   - byted-torch 2.7.1 已由系统包提供 → 本文件不重装 torch
#   - transformers 5.9.0 已由系统包提供 → 本文件不重装 transformers
#   - groundingdino 已通过 cd src/t2i-r1/src/utils/GroundingDINO && pip install -e . 装好
#   - janus 不是 PyPI 包，已 vendored 在 src/t2i-r1/src/janus，靠 PYTHONPATH 加载
#
# 安装：
#   pip install -r requirements.txt
#
# 校验：
#   bash check_train_env.sh
#
# ⚠️ flash-attn 安装失败时的回退：
#   1) 注释掉本文件最下方的 flash-attn 行
#   2) 把 src/t2i-r1/src/run_train.sh 里 --attn_implementation 改回 sdpa
#   （不必改 trainer 代码，--attn_implementation 已能正确传到 language_model）
